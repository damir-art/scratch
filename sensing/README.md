# Сенсоры
Сенсорные команды.  
Сенсорные команды помещаем в условие (control).

Команды:
- касается указатель мыши/край/спрайт
- касается цвета
- цвет касается цвета
- расстояние до
- спросить и ждать
- ответ
- клавиша нажата
- мышь нажата
- x мыши
- y мыши
- перетаскивать можно/нельзя
- громкость
- таймер
- и т.д.
